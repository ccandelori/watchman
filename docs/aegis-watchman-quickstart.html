Local agent protection

Start Watchman, point your agent at Aegis, verify protection.

The app is the normal path. Ollama stays on port 11434, Aegis listens on port 8000, and Hermes Agent uses the Aegis URL instead of the provider URL.

Agent URL http://127.0.0.1:8000/v1
Provider http://127.0.0.1:11434/v1
Model qwen3:4b
CIFT Qwen3-4B on MPS
1

Prerequisites

Have Ollama, Aegis, and the local model ready.

Use macOS with Apple Silicon for the current Qwen3-4B MPS reference path. Install Ollama, check out the Aegis repository, and run uv sync --extra dev. If qwen3:4b is not installed yet, Watchman can help pull it or you can run ollama pull qwen3:4b.

CIFT support is model-specific. Qwen3-4B is the certified reference path; other models need their own certification before you claim CIFT enforcement.
2

Open Watchman

Use the desktop app as the control surface.

From the repository root:

scripts/run-aegis-desktop-launcher.sh

For the packaged local app:

cd desktop/aegis-launcher
npm run pack:mac
open "dist/mac-arm64/Aegis Watchman.app"
3

Provider

Confirm Ollama and choose the model.

In the Provider card, Ollama should show as running on 11434. Select qwen3:4b. If Ollama is not running, click Start Ollama.

4

Protection

Click Start Protection.

Watchman checks the machine, GPU, CIFT sidecar, gateway, and console. When protection is running, the app shows the Aegis agent settings and the primary action changes to a stop action.

Expected GPU

MPS on the current Mac reference path.

Expected services

CIFT and Gateway running or adopted.

5

Agent settings

Point Hermes at Aegis, not Ollama.

Base URL: http://127.0.0.1:8000/v1
API key:  aegis-local-dev-key
Model:    qwen3:4b
Direct Ollama traffic on 11434 bypasses Watchman. Use it only as a model baseline, not while testing Aegis protection.
6

Verify

Run verification and inspect the evidence.

Click Run Verification. Then open Activity and Trace. The key proof is a CIFT block where the provider/model call is skipped before generation. Use Latency and Detectors to explain overhead and the detector path.
